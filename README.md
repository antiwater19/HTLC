*HTLC 대학교 과제*
`HTLC lock 소스코드을 활용하여 ethereum testnet에서 본인의 etherum wallet 계정에서 스마트 컨트랙트로 동작시키고, 이를 Web3.js 로 연계 호출하여 동작하는 것을 만드는 과제입니다.`
-------------------------------------------
npm으로 설치하여 사용한것들 
    npm install truffle
    npm install ganache
    npm install web3

### truffle을 실행하는법
    truffle develop
실행후 develop모드에서 
    test
또는
    truffle test
를 입력하여 실행 할 수 있다.

### ganache를 실행하는 법
    npm run ganache
를 이용하여 실행할 수 있다.
